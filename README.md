# worldcup2026live

Live scores, match schedule and group standings for FIFA World Cup 2026.

🌐 **Live site:** [worldcup2026live.site](https://worldcup2026live.site)

## Features
- Live match scores (auto-updates every 60 seconds)
- Full match schedule (104 matches)
- Group standings
- Mobile-friendly

## Setup
1. Clone the repo
2. Open `index.html` in a browser — it works directly
3. For GitHub Pages: Settings → Pages → Deploy from main branch

## Data
Powered by [football-data.org](https://football-data.org) API.
